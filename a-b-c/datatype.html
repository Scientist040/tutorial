Data type

Data type yana nufin irin data da zaa ajiye acikin variable, akwai lamba akwai rubutu da sauransu. Aduk lokacin da zaa hada variable dole a fadawa computer irin abinda za a samata ta ajiye. Akwai data type kamar su:
  1. String
  2. integer
  3. Boolean
  4. Floating point
String yana nufin rubutu kama “khalifa” ko rubutu hade da lambobi kamar “khalifa000”. Sannan shi string ana sakashi tsakanin wakafi biyu alamar maganar wani “”.
Integer yana nufin lamba ko whole number daga 0 zuwa 9 haka negative -1 ko -2.
Floating point yanan nufin lamba wacce take dauke da point kamar 9.1
Bolean yana daukar value guda biyu ne kawai true da false. Idan akace abu true ne ana nufin computer tasan da abin.
yadda ake amfani dasu tare da variable zara rubuta data type a farko sai arubuta sunan variable sannan abiyo bayansa da abinda zaa ajiye. Kamar haka: 

 string khalifa = “He is programmer”;
 int khalifa = 7676;
 float khalifa = 9.7;
 bool khalifa = true;
za a iya ganin bambanci a wasu programming language din wajen irin kalmomin da ake amfani dasu (syntax) ko a wani abun kamar a yaren JavaScript da ba a saka data type.
