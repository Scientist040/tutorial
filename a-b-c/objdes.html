Object Destructuring

Zamu fahimci wannan da wuri idan mukayi amfani da misali. Idan munasan dafa tajiya jolof shin zamu dauki duk abinda yake cikin store ne? Amsa itace aa saboda store dinmu yana dauke da kayan abinci da yawa, saboda haka zamu dauki iya abinda muke bukata ne. 
Syntax
Var = {property} = expression.

Property to variable

Da object destructuring zamu kirkiri variable ta hanyar amfani da property da value din object. Munada object na store kamar haka: 

var store = {
 name:"taliya",
 title:"shinkafa",
 kaya:"tattasai"
}
Idan muna bukatar kayan jolof zamu yishi kamar haka ne. 
var {name} = store;
Wato anan kenan name shine property na store object, shi kuma store shine sunan object din da muka ciro abin daga cikinsa. A yanzu name yazama variable me zaman kansa idan mukayi: 
console.log(name)
zamuga taliya ne value dinsa.

Multiple peoperties

Sannan zamu iya ciro properties sama da daya ta hanyar sanya comma atsakaninsu kamar haka: 

 var {name, title} = store;
Name => taliya
Title =>shinkafa

Object Properties Function Parameters

Zamu iya amfani da destructuring a funtion kamar haka: 
fun(store);
Anan nakira function din sannan nasa masa sunan object din a parameter. 

     function fun({name,kaya}){
    return data = "Sunan abinci" + name + "Kayan Hadi" +kaya;
    }
Abinda function din nan zaiyi return shine: 
Sunan Abinci taliya Kayan hadi tattasai.

Default values

Wani lokacin babu property din da muka nufi cirowa daga object, misali na rubuta Var {taruhu} = store; Kunga acikin store object babu taruhu, to anan taruhu zai nuna undefined, amma zamu yi amfani da default value mu sama masa value aciki kamar haka: 

var {taruhu = ['daya']} = store;
Anan gurin abinda yake faruwa shine idan akwai property din to zai dauki value dinsa idan babu zai dauki wanda aka sa masa misali kamar ‘daya’ da muka rubuta. Kenan dede yake da wannan: 

 var taruhu = store.taruhu == undefined ? 'daya' : store.taruhu;

Alias

Wani lokacin zamuji cewa munasan sunan variable din ya saba da sunan property din da mukaciro daga object to anan ne muke amfani da alias. 
Syntax 
Var {property:aliasIdentifier} = store;
Kenan 

var {taliya:science} = store;
Anan science shine sunan sabon variable dinmu wanda yake dauke da value din taliya property. Dede yake da wannan 

var science = store.taliya;

Rest Object

Shikuma wannnan yana bamu damar kwashe sauran properties da suka rage acikin object bayan mun ciri wani daga ciki. Zamuyi amfani da spread operator(...) anan. 
Syntax
Var {property, ...rest}= objectName;
Anan property shine sunan property da muka ciro daga cikin object din kamar de yadda mukayi a farkon darasi sannan rest shine object dinda zai riki sauran properties din da bamu ciraba daga cikin object din. 

    var {name, ...sauran} = store;

Haka abin zai kasance:
Name=> taliya
Sauran = > {title:”shinkafa”, kaya:”tattasai”}
