Call Method

Zamu iya amfani da call method muyi inheritence. 
syntax
functionName.call(this keyword, parameterst);
Yanzu mudana constructor function kamar haka: 

function con(fname,lname){
            this.fname =  fname;
            this.lname = lname;
        }
Wannan function din zai bamu damar hada object daukeda properties guda biyu wato fname da lname. Sannan munada wani constructor function da yake bamu damar hada mname. 

 function fun2(mname){
            this.mname = mname;
        }
To anan munasan musamu fname da lname da nmane, a madadin mukara wani function din na uku zamuga code din zaiyi yawa to sai muyi amfani da call() method, sannan mukara parameters dinsa mukeso mu yi inheritence din. 
     
  function fun2(mname, fname, lname){
            con.call(this, fname,lname);
            this.mname = mname;
        }
Yanzu zamu shigar da instance. 

 const ns = new fun2("khalifa","ali","ahmad")
 console.log(ns)
 // ns => ali ahmad khalifa

Prototype-based inheritance

Prototype wani property ne wanda yake kunshe acikin kowanne constructor function a JavaScript, ana iya amfani dashi wajen yin inheritence. Muduba wannan: Munada constructor function kamar haka da instance guda biyu: 


       function member(fname,lname){
            this.fname =  fname;
            this.lname = lname;
        }
     const dev = new member("khalifa","ali");
     const pro = new member("muhammad","ali");
munasan muhada function acikin dev object wanda zai bamu fname din da lname a hade, zamuyi haka: 

  function member(fname,lname){
            this.fname =  fname;
            this.lname = lname;
        }
     const dev = new member("khalifa","ali");
     const pro = new member("muhammad","ali");
    // function din da zai bamu cikakken suna
     dev.cikakkenSuna = function(){
        return this.fname + this.lname;
    }

    // zamu kira function din anan
    console.log(dev.cikakkenSuna())
